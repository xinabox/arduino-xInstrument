# arduino-xInstrument Library

Currently supported:
* Microsoft Azure
* Google sheet
* OD01 ouput
* Serial Output
* Blynk BLE

## Dependency
* [arduino core](https://github.com/xinabox/arduino-CORE)
* [arduino-OD01](https://github.com/xinabox/arduino-OD01)
* [arduino-Provision](https://github.com/xinabox/arduino-Provision)
